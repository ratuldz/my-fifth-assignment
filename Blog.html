Question-2: What is the difference between innerHTML, innerText and textconten?

In HTML, innerHTML, innerText, and textContent are properties of the DOM (Document Object Model). They allow you to read and update the content of HTML elements..

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript that allows you to attach a single event listener to a parent element to handle events on its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling directs an event to its target. It works like this: When an element (like a button) is clicked, an event is directed to the element. If an event handler is set for the element, the event handler is triggered. Then the event "bubbles up" to the elements parent..
